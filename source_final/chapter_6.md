# บทที่ 6 เทคโนโลยี BPMR

ปัจจุบันนี้การเติบโตทางด้านอินเทอร์เน็ตและการสื่อสารดิจิทัลเป็นไปอย่างรวดเร็วจึงส่งผลทำให้มี ความต้องการของพื้นที่ในการจัดเก็บข้อมูลเพิ่มมากขึ้นเรื่อยๆ ฮาร์ดดิสก์ไดรฟ์ในปัจจุบันจะใช้สื่อ บันทึกที่เคลือบด้วยฟิล์มบางของสารแม่เหล็กซึ่งประกอบด้วยเกรนเชิงแม่เหล็ก30 (magทetic grain) ขนาดเล็กในระดับนาโนเมตร โดยแต่ละเกรนจะมีสภาพความเป็นแม่เหล็ก (mลgทetizatioท) ไปใน ทิศทางที่แตกต่างกัน ดังนั้นการบันทึกข้อมูลจะอาศัยสนามแม่เหล็กจากหัวเขียน (พrite head) เพื่อทำให้เกรนหลายๆ เกรนมีสภาพความเป็นแม่เหล็กไปในทิศทางที่ต้องการ (ขนานหรือตั้งฉาก กับระนาบของสื่อบันทึก) โดยทั่วไปการเพิ่มความหนาแน่นเชิงพื้นที่ (areal deกรity) หรือความจุ ข้อมูลทำได้โดยการลดขนาดของเกรนให้เล็กลง อย่างไรก็ตามการลดขนาดของเกรนเพื่อเพิ่มความ หนาแน่นเชิงพื้นที่มีแนวโน้มที่จะทำได้ยากขึ้น เพราะเกรนที่มีขนาดเล็กมากเกินไปจะไม่มีเสถียรภาพ ในการเก็บข้อมูล เนื่องจากพลังงานความร้อนจากภายนอกจะทำให้สภาพความเป็นแม่เหล็กของ เกรนมีทิศทางเปลี่ยนไปได้ง่าย (ทำให้บิตข้อมูล "0" เปลี่ยนเป็น "1" หรือในทางตรงกันข้าม) ซึ่ง ปรากฏการณ์นี้เรียกว่า "ขีดจำกัดซูเปอร์พาราแมกเนติก31 (super-paramagnetic limit)" [1, 43] หรือขีดจำกัดความหนาแน่นเชิงพื้นที่

ดังนั้นเทคโนโลยีใหม่ที่สามารถจัดเก็บข้อมูลได้มากกว่าเทคโนโลยีการบันทึกข้อมูลแบบที ใช้ในปัจจุบัน (นั่นคือแบบแนวตั้ง) จึงเป็นสิ่งจำเป็นอย่างยิ่งในอนาคต เช่น เทคโนโลยี HAMR (heat assisted magnetic recording) [78], เnคlulaยี BPMR (bit-patterned magnetic recording) [79], และเทคโนโลยี TDMR (two-dimensional magnetic recording) [80] เป็นต้น ซึ่ง เทคโนโลยีเหล่านี้สามารถจัดเก็บข้อมูลได้มากกว่า 1 เทระบิตต่อตารางนิ้ว (terabitร/in2 หรือ Tb/in2)

ในบทนี้จะอธิบายเฉพาะพื้นฐานและหลักการทำงานของเทคโนโลยี BPMR พร้อมทั้งสรุป ภาพรวมของงานวิจัยต่างๆ ที่ผ่านมาที่เกี่ยวข้องกับเทคโนโลยี BPMR เพื่อให้ผู้อ่านสามารถใช้เป็น จุดเริ่มต้นในการทำวิจัยทางด้านระบบการประมวลผลสัญญาณของเทคโนโลยี BPMR ได้

## 6.1 บทนำ

ในระบบการบันทึกเชิงแม่เหล็ก สมรรถนะของระบบในรูปของอัตราข้อผิดพลาดของบิต (BER: biterror rate) จะขึ้นอยู่กับค่าอัตราส่วนกำลังของสัญญาณต่อกำลังของสัญญาณรบกวน (SNR: signalto-noise ratio) ซึ่งในทางปฏิบัติค่า รNR ในสัญญาณอ่านกลับจะสัมพันธุ์กับจำนวนเกรนต่อบิต เซลล์ (bit cel) หรือข้อมูลหนึ่งบิต โดยทั่วไปสัญญาณรบกวนการเปลี่ยนสถานะ (traทรitioก noise) ถือว่าเป็นสัญญาณรบกวนหลักที่พบมากในระบบการบันทึกแบบแนวตั้ง [79] ซึ่งความรุนแรงของ ขู้ซู้   
สัญญาณรบกวนการเปลี่ยนสถานะนี้ขึ้นอยู่กับการเปลี่ยนแปลงแบบซิกแซก (zig-zag traทรition) ณ บริเวณรอยต่อที่เป็นขอบเขตของแต่ละบิตเซลล์ โดยการเปลี่ยนแปลงแบบซิกแซกจะมีความ รุนแรงเมื่อเกรนแต่ละเกรนมีขนาดใหญ่ (และรุนแรงน้อยเมื่อเกรนมีขนาดเล็ก) ดังแสดงในรูปที่ 6.1 ดังนั้นถ้าสื่อบันทึกที่ใช้ประกอบด้วยเกรนขนาดใหญ่จำนวนมาก ก็จะส่งผลทำให้ระบบต้องเผชิญ กับบัญหาเรื่องสัญญาณรบกวนการเปลี่ยนสถานะที่รุนแรง เพราะฉะนั้นสื่อบันทึกที่สร้างจากเกรน ขนาดเล็กจึงเป็นสิ่งที่จำเป็นอย่างยิ่ง และหนึ่งบิตเซลล์ก็ควรประกอบด้วยเกรนขนาดเล็กจำนวนมาก เพียงพอเพื่อให้ระบบมีค่า รNR ตามที่ต้องการ

โดยทั่วไปการเพิ่มความจุข้อมูลของฮาร์ดดิสก์ไดรฟ์ทำได้โดยการลดขนาดของบิตข้อมูลให้ เล็กลง แต่ยังคงรักษาระดับ รNR ให้เท่าเดิม หรือกล่าวอีกนัยหนึ่งคือการลดขนาดของเกรนให้ เล็กลง แต่ยังคงมีจำนวนเกรนต่อหนึ่งบิตเซลล์คงเดิม อย่างไรก็ตามขนาดของเกรนไม่ควรเล็กเกินไป เพราะพลังงานความร้อนจากภายนอกจะสามารถเปลี่ยนทิศทางสภาพความเป็นแม่เหล็กของเกรน ได้ (ทำให้ไม่มีเสถียรภาพในการจัดเก็บข้อมูล) นั่นคือทำให้เกิดปัญหาเรื่องซูเปอร์พาราแมกเนติก [1, 43] ซึ่งเป็นขีดจำกัดที่ทำให้เทคโนโลยีการบันทึกเชิงแม่เหล็กที่ใช้ในปัจจุบันไม่สามารถเพิ่ม ความหนาแน่นเชิงพื้นที่ได้มากกว่า 1 $\mathrm { T b } / \mathrm { i n } ^ { 2 }$ ในทางปฏิบัติปัญหาเรื่องซูเปอร์พาราแมกเนติก สามารถป้องกันได้โดยการใช้สื่อบันทึกที่มีค่าอัตราส่วน [1, 43]

$$
\frac { K _ { u } V } { k _ { B } T } \ge \beta\tag{6.1}
$$

เมื่อ $K _ { u }$ คือค่าสัมประสิทธิ์แอนไอโซทรอปีแบบแกนเดี่ยวของวัสดุ, V คือปริมาตรของเกรนที่ใช้ เก็บข้อมูลหนึ่งบิต, $k _ { B }$ คือค่าคงตัวโบลตซ์แมน (Boltzmann's constant) มีค่าเท่ากับ $1 . 3 8 \times 1 0 ^ { - 2 3 }$ จูลต่อเคลวิน (joนles per kelvin), T คืออุณหภูมิสัมบูรณ์มีหน่วยเป็นเคลวิน, และ β คือจำนวน เต็มบวกที่มีค่ามากๆ เช่น $\beta = 6 0$ เป็นต้น เนื่องจากค่า $k _ { B } T$ เป็นค่าคงตัว ดังนั้นเพื่อชดเชยปริมาตร ของเกรนที่ลดลง สื่อบันทึกจะต้องใช้สารแม่เหล็กที่มีค่า สู $K _ { u }$ สูงมากขึ้นซึงผลที่ตามมาก็คือหัวเขียน ซู จะต้องใช้สนามแม่เหล็กที่มีความเข้มสูงมากในการเขียนบิตข้อมูลลงไปจัดเก็บในสื่อบันทึกได้อย่างมี เสถียรภาพ ปัญหานี้แก้ไขได้โดยใช้เทคโนโลยี HAMR [78] (ดูรายละเอียดในบทที่ 8) แต่อย่างไร ก็ตามสื่อบันทึกที่ใช้ในปัจจุบัน (ที่เคลือบด้วยฟิล์มบางของสารแม่เหล็ก) ก็กำลังจะถึงขีดจำกัดของ การจัดเก็บข้อมูลในอนาคตอันใกล้นี้

เทคโนโลยี BPMR ถือว่าเป็นเทคโนโลยีที่สามารถเพิ่มความหนาแน่นเชิงพื้นที่ได้มากกว่า 1 Tb/in2 และไม่มีปัญหาเรื่องขีดจำกัดซูเปอร์พาราแมกเนติก [79] โดยเทคโนโลยี BPMR จะใช้ สื่อบันทึกที่มีการทำแบบ (patterned media) หรือสื่อบันทึกที่มีรูปแบบแน่นอน ซึ่งสร้างจากสาร แม่เหล็กที่มีโดเมนแม่เหล็กเดี่ยว (single magnetic domain) หรือมีลักษณะเป็นไอแลนด์เชิง แม่เหล็ก (magnetc iรland) ขนาดนาโนเมตร เรียงตัวกันบนแผ่นรองรับที่ทำมาจากวัสดุทีไม่เป็น แม่เหล็ก (non-magnetic material) อย่างเป็นระเบียบและมีระยะห่างสมำเสมอ โดยทิศทางของ สภาพความเป็นแม่เหล็กของไอแลนด์อาจจะขนานหรือตั้งฉากกับแผ่นรองรับก็ได้ รูปที่ 6.2 แสดง ตัวอย่างการจัดเก็บข้อมูลในสื่อบันทึกของเทคโนโลยี BPMR

เนื่องจากไอแลนด์เชิงแม่เหล็กขนาดนาโนแบบเอกเทศ (isolated nano-scale magnetic islands) ที่ถูกล้อมรอบด้วยวัสดุที่ไม่เป็นแม่เหล็กจะทำหน้าที่เป็นแบบ "โดเมนเดี่ยว (single domain)" หรือทำหน้าที่สมือนเป็นไอแลนด์แบบเกรนเดียว32 ดังนั้นการใช้เกรนขนาดเล็กจำนวนมากต่อหนึ่ง บิตข้อมูลจึงเป็นสิ่งที่ไม่จำเป็นสำหรับเทคโนโลยี BPMR นั่นคือไอแลนด์หนึ่งไอแลนด์อาจสร้างจาก เกรนเพียงเกรนเดียวก็ได้ นอกจากนี้การใช้ไอแลนด์เชิงแม่เหล็กแบบโดเมนเดี่ยวยังสามารถจัดการ กับปัญหาเรื่องขีดจำกัดซูเปอร์พาราแมกเนติก และช่วยลดปัญหาเรื่องสัญญาณรบกวนการเปลี่ยน สถานะได้อีกด้วย [81]

รูปที่ 6.2 ลักษณะการจัดเก็บข้อมูลในสื่อบันทึกของเทคโนโลยี BPMR

ในเทคโนโลยี BPMR ตำแหน่งของไอแลนด์ทั้งหมดจะถูกกำหนดไว้เรียบร้อยแล้วตั้งแต่ ขั้นตอนการทำแบบ (หรือการสร้าง) ของสื่อบันทึก จึงทำให้มีข้อดีจำนวนมากเมื่อนำมาใช้งานจริง เช่น ลดปัญหาเรื่องสัญญาณรบกวนการเปลี่ยนสถานะ, กำจัดปัญหาเรื่องการเลื่อนบิตแบบไม่เป็น เชิงเส้น (nonlinear bit shift),ทำให้กระบวนการกู้ข้อมูลง่ายขึน, และทำให้กระบวนการติดตาม (tracking) ที่ใช้ในระบบเซอร์โว (servo system) ง่ายขึ้น [81, 82] เป็นต้น

## 6.2 วิวัฒนาการของเทคโนโลยี BPMR

หัวข้อนี้จะกล่าวถึงวิวัฒนาการของเทคโนโลยี BPMR ตั้งแต่เริ่มแรกจนถึงปัจจุบัน เพื่อให้ผู้อ่าน ทราบถึงงานวิจัยด้านต่างๆ ที่น่าสนใจที่มีส่วนช่วยทำให้เทคโนโลยี BPMR สามารถนำมาใช้งานจริง ในอนาคตอันใกล้นี้ได้

## 6.2.1 สื่อบันทึก

ในช่วงแรกงานวิจัยทางด้านเทคโนโลยี BPMR จะเน้นไปที่การพัฒนาหรือการสร้างสื่อบันทึกที่มี การทำแบบ แนวคิดในการทำแบบของสื่อบันทึกเริ่มมาตั้งแต่ปี ค.ศ. 1963 [83] โดยมีการเสนอให้ ใช้ "แทร็กที่มีการทำแบบ (patterทed track)" เพื่อลดปัญหาที่เกิดจากสัญญาณรบกวนและกระบวน การติดตามในระบบเซอร์โว จากนั้นในปี ค.ศ. 1987 ก็ได้มีการศึกษาคุณสมบัติของการใช้ "แทร็ก แบบแคบ33 (narrow tracks)" [84] และได้ทำการศึกษาและทดลองบันทึกข้อมูลลงในสื่อบันทึกที่ มีการทำแบบขนาดนาโนในปี ค.ศ. 1991 [85]

![](images/chapter_6/6d34c0bb9e3e2ea5af88c1a7bbfa4a1a42ef48cb1d5ac63981152802af7e00ee.jpg)

ตั้งแต่ปี ค.ศ. 1991 เป็นต้นมา งานวิจัยทางด้านสื่อบันทึกที่มีการทำแบบได้เน้นไปที่การ สร้างและการศึกษาคุณสมบัติของสื่อบันทึก โดยเทคโนโลยีต่างๆ ได้ถูกพัฒนาขึ้นมาเพื่อใช้สร้าง สื่อบันทึกที่สามารถจัดเก็บข้อมูลได้มาก เช่น นาโนลิโทกราฟี (nano lithography), การจัดเรียง ตัวเอง (self-assembly), สิโทกราฟีด้วยลำอิเล็กตรอu (electron beam lithography), และลิโท กราฟีด้วยการกดพิมพ์ระดับนาโน (nano-imprint lithography) เป็นต้น ผู้สนใจทางด้านการสร้าง สื่อบันทึกที่มีการทำแบบสามารถศึกษารายละเอียดเพิ่มเติมได้จาก [86 – 89]

## 6.2.2 ระบบการบันทึกเชิงแม่เหล็กสำหรับ BPMR

โดยทั่วไประบบการบันทึกเชิงแม่เหล็กแบบ BPMR แบ่งออกได้เป็น 2 ประเภทหลักคือ ระบบ BPMR แบบแผ่นหมุน (spinning-disk system) และระบบ BPMR แบบโพรบ (probe-based system) [90] ดังแสดงในรูปที่ 6.3 และ 6.4 ตามลำดับ

![](images/chapter_6/d3f1aab066a2dde91ec9213f6550fc18d9b1cb7c927a82c23420686f9764c1dc.jpg)  
รูปที่ 6.4 ระบบ BPMR แบบโพรบ [90]  
รูปที่ 6.3 แสดงระบบ BPMR แบบแผ่นหมุนที่ใช้หัวแม่เหล็กแบบ MR (mลgnetoresistivย) ร่วมกับสื่อบันทึกที่มีการทำแบบ ซึ่งมีลักษณะการทำงานคล้ายกับระบบการบันทึกแบบ แนวตั้งที่ใช้กันในปัจจุบัน ดังนั้นเทคโนโลยีต่างๆ ที่ใช้ในระบบการบันทึกแบบแนวตั้งจึงสามารถ นำมาประยุกต์ใช้กับระบบ BPMR แบบแผ่นหมุนนี้ได้ อย่างไรก็ตามกระบวนการเข้าจังหวะ (synchronizลt่on) ระหว่างตำแหน่งของหัวแม่เหล็กและตำแหน่งของไอแลนด์ที่จะทำการเขียนหรืออ่าน ข้อมูลในระบบ BPMR จะแตกต่างจากกระบวนการเข้าจังหวะที่ใช้ในระบบการบันทึกแบบแนวตั้ง ค่อนข้างมาก [91] งานวิจัยใน [91, 92] ได้ศึกษากระบวนการเขียนข้อมูลในระบบ BPMR ซึ่ง แสดงให้เห็นว่าสนามไฟฟ้าเขียน (พrite fied) จำเป็นอย่างยิ่งที่จะต้องเข้าจังหวะกับตำแหน่งของ ไอแลนด์ที่ต้องการเขียนบิตข้อมูลลงไป กล่าวคือเมื่อหัวเขียนเคลื่อนที่มาอยู่ณ ตำแหน่งจุดศูนย์กลาง ของไอแลนด์ สนามไฟฟ้าเขียนต้องถูกป้อนเข้าไปในไอแลนด์ทันทีภายในระยะเวลาที่กำหนดให้ เท่านั้น [16, 93, 94] ดังนั้นกระบวนการเข้าจังหวะที่ใช้ในระบบ BPMR ต้องการความเที่ยงตรง สูงมากกว่าระบบการบันทึกแบบแนวตั้ง นอกจากนี้วิธีในการเข้าจังหวะการเขียนอีกวิธีหนึ่งซึ่งใช้ งานได้ดีก็คือ "การอ่านในขณะเขียน (read while write)" [90] มิฉะนั้นอาจก่อให้เกิดปัญหา ข้อผิดพลาดในการเขียน (พritten-in error) [91, 95] ซึ่งมีผลทำให้บิตข้อมูลที่ต้องการเขียนเข้าไป ในไอแลนด์เกิดปัญหา34เรื่อง การแทนที่ (sub-stitนtion), การตัด (deletion) และการแทรก (insertion) นอกจากนี้งานวิจัยใน [96] ได้ศึกษาแบบข้อมูล (data pattern) ที่ใช้ในระบบเซอร์โว

พบว่าการใช้สื่อบันทึกที่มีการทำแบบจะช่วยทำให้สามารถใช้แบบข้อมูลที่ซับซ้อนในระบบเซอร์โวได้ ซึ่งมีผลทำให้ความแปรปรวนของสัญญาณข้อผิดพลาดของตำแหน่ง (PES: position error signal) ลดลงอย่างมากในระบบ BPMR

ระบบการบันทึกเชิงแม่เหล็กสำหรับ BPMR อีกแบบหนึ่งก็คือ ระบบ BPMR แบบโพรบ ตามรูปที่ 6.4 โดยหัวแม่เหล็กหลายๆ หัวจะถูกติดตั้งอยู่บนระนาบที่เคลื่อนที่อยู่เหนือสื่อบันทึก จากนั้นการเขียนข้อมูลทำได้โดยการป้อนสนามแม่เหล็กเข้าไปยังตำแหน่งของไอแลนด์ที่ต้องการ โดยสนามแม่เหล็กที่ป้อนเข้าไปจะต้องไม่ไปรบกวนไอแลนด์ตำแหน่งอื่นๆ ในทำนองเดียวกันการ อ่านข้อมูลทำได้โดยการใช้วิธีการที่เหมือนกับวิธี MFM (magnetic force microscope) [97] โดย หัวแม่เหล็กจะตรวจหาเพียงว่าไอแลนด์นั้นมีสถานะของสภาพความเป็นแม่เหล็กในทิศทางที่พุ่งขึ้น .3 หรือพุ่งลงเท่านั้น นอกจากนี้การอ่านข้อมูลในระบบ BPMR แบบโพรบยังสามารถทำได้โดยการวัด สนามแม่เหล็กโดยตรงจากไอแลนด์ โดยการใช้หัวอ่านแบบ MR ขนาดจิ๋วหรือโพรบเชิงแสง (Optical probe) เป็นต้น

ระบบ BPMR จะใช้งานได้ดีก็ต่อเมื่อระบบมีค่า รSNR ที่เพียงพอ ถึงแม้ว่าการใช้ไอแลนด์ เชิงแม่เหล็กแบบโดเมนเดี่ยวสามารถช่วยลดผลกระทบที่เกิดจากสัญญาณรบกวนการเปลี่ยนสถานะ และการเลื่อนบิตแบบไม่เป็นเชิงเส้น [81, 82] แต่ความไม่สมบูรณ์ในการสร้างสื่อบันทึกจะทำให้ เกิดสัญญาณรบกวนสือบันทึก (media ทoise) รูปแบบใหม่[98, 99] ซึ่งมีผลทำให้สมรรถนะของ ระบบ BPMR ลดลงอย่างมาก [100] ซึ่งจากการศึกษาพบว่าสัญญาณรบกวนสื่อบันทึกนี้เกิดจาก การผันผวนแปรผัน (fuctนation) ของขนาดและตำแหน่งของไอแลนด์เชิงแม่เหล็ก [99 – 103]

## 6.2.3 การประมวลผลสัญญาณในระบบ BPMR

ในช่วงแรกของการพัฒนาระบบการประมวลผลสัญญาณในระบบ BPMR นักวิจัยจะเน้นไปที่ระบบ BPMR แบบแผ่นหมุน ซึ่งมีลักษณะการทำงานคล้ายกับระบบการบันทึกข้อมูลแบบที่ใช้กันทั่วไป (นั่นคือแบบแนวนอนและแบบแนวตั้ง) ดังนั้นช่องสัญญาณอ่านของระบบ BPMR จึงสามารถ จำลองเป็นแผนภาพบล็อกแบบง่ายได้ตามรูปที่ 6.5 [10] ซึ่งประกอบด้วยช่องสัญญาณ (chaทnel), อีควอไลเซอร์แบบผลตอบสนองบางส่วน (partial-response eqนalizer), และวงจรตรวจหาวีเทอร์บิ (Viterbi detector) [13]

งานวิจัยทางด้านช่องสัญญาณอ่านของระบบ BPMR ได้เริ่มต้นในปี ค.ศ. 1998 [102] โดยใช้ช่องสัญญาณผลตอบสนองบางส่วนควรจะเป็นสูงสุด (PRML: partial-response maximumlikelihoอd) ในการวัดสมรรถนะของระบบ BPMR โดยใช้ไอแลนด์ที่มีสภาพความเป็นแม่เหล็ก 64ส แบบแuวuอน (ongitudinal magnetization) จากนันงานวิจัยใน [81, 100, 101] ได้ศึกษา กระบวนการอ่านของระบบ BPMR ที่ใช้สื่อบันทึกสำหรับไอแลนด์ที่มีสภาพความเป็นแม่เหล็ก

![](images/chapter_6/7c9364306d5fa50532d098f3885ad2d4f45b5778afcaa0957436e9aba99ca409.jpg)  
รูปที่ 6.5 แผนภาพบล็อกแบบง่ายสำหรับช่องสัญญาณอ่านของระบบ BPMR

aซ แบบแนวนอน นอกจากนี้งานวิจัยใน [100, 101] ได้ศึกษาช่องสัญญาณอ่านของระบบ BPMR ทีใช้ สื่อบันทึกสำหรับไอแลนด์ที่มีสภาพความเป็นแม่เหล็กแบบแนวตั้ง (perpeทdicนlar magnetization) โดยสัญญาณอ่านกลับ (readback signal) จะถูกจำลองจากแนวคิดแบบสองมิติของการคำนวณ อินทิกรัลการตอบสนอง (reciprocity integral)

เพื่อให้แบบจำลองของสัญญาณอ่านกลับมีความถูกต้องมากยิ่งขึ้น งานวิจัยใน [104] ได้ เสนอแนวคิดแบบสามมิติของการคำนวณอินทิกรัลการตอบสนองเพื่อประมาณค่าผลตอบสนอง สัญญาณพัลส์ (pulse reรponรe) ของระบบ BPMR และได้ทำการศึกษาผลกระทบของรูปทรงของ ไอแลนด์ (island geometry) ที่มีต่อผลตอบสนองสัญญาณพัลส์นี้ ซึ่งพบว่าผลตอบสนองสัญญาณ พัลส์ของระบบ BPMR จะขึ้นอยู่กับขนาดและรูปร่างของไอแลนด์ค่อนข้างมาก นอกจากนี้งานวิจัย ใน [105, 106] ได้ศึกษาผลกระทบของรูปทรงของไอแลนด์ต่อสมรรถนะของช่องสัญญาณอ่านใน รูปของอัตราข้อผิดพลาดของบิต (BER) โดยงานวิจัยนี้ได้ใช้การคำนวณอินทิกรัลการตอบสนอง แบบสามมิติ (3D reciprocity interal) ในการประมาณค่าผลตอบสนองสัญญาณพัลส์ของระบบ BPMR และสัญญาณอ่านกลับที่ได้จากหัวอ่านเกิดจากการซ้อนทับเชิงเส้น (linear superpoรition) ของผลตอบสนองสัญญาณพัลส์ของแทร็กหลัก (main track) และแทร็กข้างเคียง (adjacent track) หรืออาจกล่าวได้ว่าสัญญาณอ่านกลับที่ได้จากหัวอ่านมีผลกระทบที่เกิดจากแทร็กข้างเคียงหรือที เรียกว่า "การแทรกสอดระหว่างแทร็ก (ITI: inter-track interference)" ซึ่งจากการทดลองพบว่า การแทรกสอดระหว่างแทร็กมีผลทำให้สมรรถนะของระบบ BPMR ด้อยลงมาก อย่างไรก็ตาม การลดผลกระทบของการแทรกสอดระหว่างแทร็กสามารถทำได้โดยการใช้อีควอไลเซอร์สองมิติ (2D equalizer) [107] และวงจรตรวจหาวีเทอร์บิสองมิติ (2D Viterbi detector) [108, 109] ใน การตรวจหาข้อมูล (ดูรายละเอียดในบทที7)

นอกจากนี้ปัญหาเรื่องออฟเซ็ตของหัวอ่าน (read head offset) หรือแทร็กมิสเรจิสเตรชัน (TMR: track mis-registration) รวมทั้งความไม่แน่นอนของไอแลนด์เชิงแม่เหล็กในสื่อบันทึกก็มี ผลทำให้สมรรถนะของระบบ BPMR ลดลงด้วย [110] โดยทั่วไประบบ BPMR ที่ใช้สื่อบันทึกที่มี ไอแลนด์เชิงแม่เหล็กเรียงตัวแบบกริดเฮกซะโกนัล (hexagonal grid) จะมีสมรรถนะดีกว่าระบบ BPMR ที่ใช้สื่อบันทึกที่มีไอแลนด์เชิงแม่เหล็กเรียงตัวแบบกริดมุมฉาก (rectangular grid) เพราะว่า การเรียงตัวแบบกริดเฮกซะโกนัลจะทำให้ไอแลนด์ของแทร็กที่อยู่ติดกันมีระยะห่างมากกว่าการเรียง ตัวแบบกริดมุมฉากสำหรับระยะแทร็ก35(track pitch) ที่เท่ากัน จึงส่งผลให้การแทรกสอดระหว่าง แทร็กลดลง รูปที่ 6.6 แสดงลักษณะการเรียงตัวของไอแลนด์เชิงแม่เหล็กแบบกริดมุมฉากและแบบ กริดเฮกซะโกนัล

รูปที่ 6.6 การเรียงตัวของไอแลนด์เชิงแม่เหล็กแบบ (ก) กริดมุมฉาก และ (ข) กริดเฮกซะโกนัล

สัญญาณรบกวนสื่อบันทึก (media noise) ในระบบการบันทึกแบบ BPMR จะแตกต่าง จากระบบการบันทึกแบบที่ใช้กันทั่วไป กล่าวคือระบบ BPMR จะไม่มีปัญหาเรื่องสัญญาณรบกวน การเปลี่ยนสถานะ (transition noise) หรือการเลื่อนบิตแบบไม่เชิงเส้น (nonlinear bit shift) เพราะว่าบิตข้อมูลถูกจัดเก็บในกลุ่มไอแลนด์เชิงแม่เหล็กแบบโดเมนเดียว และตำแหน่งของกลุ่ม ไอแลนด์เชิงแม่เหล็กก็ถูกกำหนดไว้อย่างแน่นอน อย่างไรก็ตามสัญญาณรบกวนสื่อบันทึกในระบบ BPMR จะเกิดจากความไม่สมบูรณ์ในการผลิตกรรม (fabrication imperfection) ซึ่งมีผลทำให้เกิด ความผันผวนของตำแหน่ง ขนาด ความสูง และรูปร่าง ของแต่ละไอแลนด์ โดยสัญญาณรบกวน สื่อบันทึกนี้ถือเป็นสัญญาณรบกวนหลักที่พบในระบบ BPMR ซึ่งจะทำให้สมรรถนะของระบบ ด้อยลงอย่างมาก [102, 110, 111]

งานวิจัยต่างๆ ที่กล่าวมาข้างต้นจะอยู่บนสมมุติฐานที่ว่าหัวแม่เหล็ก (หัวอ่านและหัวเขียน) มีขนาดที่เหมาะสมกับขนาดของไอแลนด์เชิงแม่เหล็กในระบบ BPMR อย่างไรก็ตามหัวแม่เหล็กที่ใช้ ในปัจจุบันจะมีขนาดใหญ่กว่าขนาดของไอแลนด์ค่อนข้างมาก36 งานวิจัยใน [112] ได้นำเสนอการ ใช้หัวแม่เหล็กที่มีขนาดใหญ่กว่าไอแลนด์เชิงแม่เหล็กสำหรับระบบ BPMR ในยุคเริ่มต้นตามที่แสดง ในรูปที่ 6.7 นอกจากนี้ยังได้เสนอแบบจำลองช่องสัญญาณอ่านสำหรับ "หลายไอแลนด์ต่อหนึ่ง หัวอ่าน (multiple islands per read head)" โดยสัญญาณอ่านกลับจะเป็นฟังก์ชันของสภาพความ เป็นแม่เหล็กของบิตข้อมูลในแทร็กที่ติดกันสามแทร็ก (แทร็กบน แทร็กหลัก และแทร็กล่าง) นั้นคือ แทร็กบนและแทร็กล่างเป็นสาเหตุของการแทรกสอดระหว่างแทร็ก (IT1) อย่างไรก็ตามวงจรภาครับ จะทำหน้าที่ในการถอดรหัสข้อมูลของแทร็กหลักเท่านั้น (ไม่สามารถถอดรหัสข้อมูลในแทร็กบน และแทร็กล่างได้ เพราะแบบข้อมูลหลายแบบที่เกิดจากแทร็กทั้งสามแทร็กอาจทำให้ได้เป็นสัญญาณ อ่านกลับแบบเดียวกันได้ [108] จึงทำให้ยากต่อการถอดรหัสข้อมูลทั้งสามแทร็กได้อย่างถูกต้อง)

## 6.3 ผลตอบสนองสัญญาณพัลส์ของระบบ BPMR

ระบบการบันทึกแบบแนวตั้งที่ใช้ในปัจจุบัน ระยะแทร็ก (track pitch) มีค่ามากกว่าคาบเวลาของบิต (bit period) ค่อนข้างมาก จึงทำให้ไม่มีปัญหาเรื่องการแทรกสอดระหว่างแทร็ก (IT) อย่างไรก็ตาม ถ้าต้องการให้ระบบ BPMR มีความหนาแน่นเชิงพื้นที่มากกว่าหรือเท่ากับ 1 Tb/in2 แต่ละไอแลนด์ เชิงแม่เหล็กต้องมีระยะห่างระหว่างกันไม่เกิน 25 นาโนเมตร (nm) ดังนันระบบ BPMR จะต้อง มีระยะแทร็กใกล้เคียงกับคาบเวลาของบิต จึงทำให้ต้องเผชิญกับปัญหาเรื่องการแทรกสอดระหว่าง สัญลักษณ์ (IS1) และ ITI ซึ่งเรียกรวมกันว่า "การแทรกสอดแบบสองมิติ (2D interfereทce)"

นอกจากนี้การเกิดสัญญาณรบกวนสื่อบันทึกในระบบ BPMR จะแตกต่างจากในระบบ การบันทึกแบบแนวตั้ง [113] กล่าวคือสัญญาณรบกวนสื่อบันทึกในระบบ BPMR เป็นผลมาจาก ความผันผวนของขนาดและตำแหน่งของไอแลนด์เชิงแม่เหล็กในสองทิศทางคือ แนวตามแทร็ก และแนวขวางแทร็ก ซึ่งมีผลทำให้ "สัญญาณพัลส์ตามแทร็ก (along-track pulse)" และ "โพรไฟล์ ขวางแทร็ก (cross-track profile)" เปลี่ยนแปลงไป ในทางปฏิบัติการแทรกสอดแบบสองมิติและ สัญญาณรบกวนสื่อบันทึกจะมีผลกระทบอย่างมากต่อสมรรถนะของระบบ BPMR ในรูปของอัตรา ข้อผิดพลาดของบิต [110, 114]

ในหัวข้อนี้จะอธิบายการสร้างผลตอบสนองสัญญาณพัลส์แบบสองมิติของช่องสัญญาณ BPMR เพื่อนำมาใช้ในการจำลองการแทรกสอดแบบสองมิติและสัญญาณรบกวนสื่อบันทึกที่ เหมือนจริงกับที่พบในระบบ BPMR โดยในการสร้างผลตอบสนองสัญญาณพัลส์แบบสองมิตินี้จะ อาศัยการคำนวณอินทิกรัลการตอบสนองแบบสามมิติ (3D reciprocity integral)เทียบกับศักย์ (poteทtial) ของหัวอ่านแบบ MR [108] จากนันจะทำการศึกษาผลกระทบของรูปทรงของหัวอ่าน และไอแลนด์ที่มีต่อผลตอบสนองสัญญาณพัลส์แบบสองมิตินี้

## 6.3.1 การจำลองผลตอบสนองสัญญาณพัลส์แบบสองมิติ

การจำลองผลตอบสนองสัญญาณพัลส์แบบสองมิติในหัวข้อนี้จะอยู่บนสมมุติฐานของสื่อบันทึก เชิงแม่เหล็กแบบแนวตั้ง (perpendicนlar magnetic media) และหัวอ่านแบบ MR เท่านั้น

สำหรับหัวอ่านแบบ MR แรงดันไฟฟ้าอ่านกลับ (readback voltage) จะเป็นสัดส่วนกับ ฟลักซ์สัญญาณ (signal flux) ที่ใส่เข้าไปในชิ้นส่วน MR (MR element) ณ บริเวณพื้นผิวของ ABS (air-bearing surface) ซึ่งเขียนเป็นสมการคณิตศาสตร์ได้คือ

$$
V _ { \scriptscriptstyle M R } \left( x , z \right) = C \phi \left( x , z \right)\tag{6.2}
$$

เมื่อ C คือค่าคงตัว, $V _ { M R }$ คือแรงดันไฟฟ้าอ่านกลับ, $\phi$ คือฟลักซ์สัญญาณ, x คือทิศทางในแนว ตามแทร็ก, และ z คือทิศทางในแนวขวางแทร็ก รูปที่ 6.8 แสดงรูปทรงของหัวอ่านแบบ MR และ ไอแลนด์เชิงแม่เหล็กแบบจัตุรัส (รqนare isโand) เมื่อ a คือความยาวของไอแลนด์, 8 คือความสูง ของไอแลนด์, d คือระยะบินของหัวอ่าน (fly height), g คือความกว้างของช่องว่าง (gap) ระหว่าง ฉนวน (shield) และชิ้นส่วน MR, W คือความกว้างของชิ้นส่วน MR, และ t คือความหนาของ ซึ้นส่วน MR

อาศัยหลักการตอบสนอง (reciprocity principle) ฟลักซ์สัญญาณที่ใส่เข้าไปในชิ้นส่วน MR ณ บริเวณพื้นผิวของ ABS สามารถเขียนได้เป็น [108]

$$
\phi \left( x , z \right) = \mu _ { 0 } \intop \displaylimits _ { - \infty } ^ { \infty } \intop \int \displaylimits _ { - \infty } ^ { + \delta } \frac { { { H _ { y } } \left( { { x } ^ { \prime } } , { { y } ^ { \prime } } , { { z } ^ { \prime } } \right) } } { i } { { M _ { y } } \left( { { x } ^ { \prime } } - { { x } , { y } ^ { \prime } } , { { z } ^ { \prime } } - z \right) { { d } { x } ^ { \prime } } } { { d { y } ^ { \prime } } } d { z } ^ { \prime }\tag{6.3}
$$

เมื่อ $\mu _ { 0 }$ คือสภาพให้ซึมผ่านได้ (permeability) ของปริภูมิเสรี (free space), i คือกระแสในขดลวด จินตภาพ (imaginary coil), $H _ { y }$ คือสนามของหัวอ่านที่เกิดจากขดลวดจินตภาพ, $M _ { y } ( x , y , z )$ คือ สภาพความเป็นแม่เหล็กของสื่อบันทึก (mediนm magnetization), และ y คือทิศทางในแนวตั้งฉาก กับสื่อบันทึกตามรูปที่ 6.8 เนื่องจากสนามแม่เหล็ก (magnetic field) คือเกรเดียนต์ของศักย์แม่เหล็ก (magnetic potential) ดังนั้นสำหรับสภาพความเป็นแม่เหล็กแบบแนวตั้ง อินทิกรัลการตอบสนอง สามารถแสดงให้อยูในรูปของศักย์สนามแม่เหล็ก (magnetic field potential) นั้นคือ $\Psi \left( x , y , z \right)$

หรือ

$$

หลักการตอบสนองบอกให้ทราบว่าฟลักซ์สัญญาณสามารถแสดงให้อยู่ในรูปของการทำคอนโวลูชัน ระหว่างศักย์สนามแม่เหล็กของหัวอ่านแบบ MR และประจุแม่เหล็กของสื่อบันทึก (mediนm magnetic charge) [43]

$$

เมื่อ $\Psi _ { s }$ คือศักย์แม่เหล็กบนพื้นผิว (SMP: surface magnetic potential) ของหัวอ่าน โดยที่ศักย์ แม่เหล็กบนพื้นผิวสำหรับครึ่งหนึ่งของความกว้างของชิ้นส่วน MR หาได้จาก [115]

$$

$$

รูปที่ 6.10 รูปคอนทัวร์ของศักย์แม่เหล็กบนพื้นผิว (SMP) ของหัวอ่านแบบ MR ในรูปที่ 6.9

เพียงครึ่งส่วนของI เท่านั้น ดังนั้นถ้าสมมุติว่า Iมีความสมมาตร อีกครึ่งส่วนของ Iก็คือ ส่วนที่เป็นภาพสะท้อนของ $\Psi _ { s } ^ { \mathrm { h a l f } }$ ซึ่งเมื่อนำเอา Ihalและส่วนที่เป็นภาพสะท้อนของ $\Psi _ { s } ^ { \mathrm { h a l f } }$ มา ประกบกัน ก็จะได้เป็นค่า I, ซึ่งมีลักษณะเป็นเมทริกซ์ที่มีสมาชิกส่วนใหญ่ในแถวกลางเป็นค่าหนึ่ง (ซึ่งแสดงถึงค่าศักย์แม่เหล็กบนหัวอ่าน) เนื่องจากศักย์แม่เหล็กบนหัวอ่านมีค่าเท่ากับหนึ่งเสมอ ดังนั้นให้ทำสำเนาข้อมูลในแถวกลางนี้ แล้วแทรกเพิ่มเข้าไปในเมทริกช์รอบแถวกลางเดิมหลายๆ แถวตามจำนวนที่สอดคล้องกับความหนาของชินส่วน MR

ศักย์แม่เหล็กในสมการ (6.6) สามารถหาได้จากค่า รMP ในสมการ (6.7) โดยใช้อินทิกรัล เชิงตัวเลข (numerical integral) นอกจากนี้ในการจำลองชั้น SUL (soft magnetic underlayer) ของสื่อบันทึกแบบแนวตั้ง จะสมมุติว่าชั้น รบL เป็นแบบสมบูรณ์ที่มีความหนาเป็นค่าอนันต์ ถ้าให้ หัวอ่านมีภาพสมบูรณ์ (perfect image) จะได้ว่าฟังก์ชันความไวรวม (total sensitivity function) ประกอบด้วยฟังก์ชันความไวของหัวอ่านและฟังก์ชันความไวของภาพหัวอ่าน (read-head's image) เมื่อเทียบกับขอบของชั้น SUL [116] ดังนั้นฟลักซ์แม่เหล็กรวมทีใส่เข้าไปในชิ้นส่วน MR ที่พื้นผิว ของ ABS สามารถเขียนเป็นสมการคณิตศาสตร์ได้คือ

$$
\phi ( x , z ) = \frac { \mu _ { 0 } } { i } \intop \intop \intop \intop \limits _ { - \infty } ^ { \infty } \intop \limits _ { d } ^ { + \delta } \intop \Psi _ { \mathrm { t o t a l } } ( x ^ { \prime } , y ^ { \prime } , z ^ { \prime } ) \times [ \frac { \partial M _ { y } ( x ^ { \prime } - x , y ^ { \prime } , z ^ { \prime } - z ) } { \partial y ^ { \prime } } ] \} d x ^ { \prime } d y ^ { \prime } d z ^ { \prime }\tag{6.8}
$$
![](images/chapter_6/1d28c62685a11b7e4a28b9bd7a80e0c914c66c00036772d60176d07a70aea03b.jpg)  
รูปที่ 6.9 ศักย์แม่เหล็กบนพื้นผิวของหัวอ่านแบบ MR ที่มี $t = 4 ~ \mathrm { n m }$ , W = 20 nm, และ $g = 1 0$ nm

เมื่อ $\Psi _ { s }$ คือศักย์แม่เหล็กบนพื้นผิว (SMP: surface magnetic potential) ของหัวอ่าน โดยที่ศักย์ แม่เหล็กบนพื้นผิวสำหรับครึ่งหนึ่งของความกว้างของชิ้นส่วน MR หาได้จาก [115]

$$
\Psi _ { s } ^ { \mathrm { h a l f } } = 1 - \left( \frac { 1 } { \pi } \right) \arctan \left( \frac { \sqrt { 2 } \sqrt { K - 1 + \exp \left( \displaystyle \frac { 2 \pi z } { \tilde { g } } \right) } \cos \left( \displaystyle \frac { 2 \pi x } { \tilde { g } } \right) } { 1 - K } \right)\tag{6.7}
$$

เมื่อ

$$
K = \sqrt { 1 - 2 \exp \left( \frac { 2 \pi z } { \tilde { g } } \right) \cos \left( \frac { 2 \pi x } { \tilde { g } } \right) + \exp \left( \frac { 4 \pi z } { \tilde { g } } \right) }
$$

และ $\tilde { g }$ คือระยะห่างระหว่างฉนวนทั้งสอง (shield-to-shield spacing) ซึ่งจากในกรณีนี้จะได้ว่า $\tilde { g } \approx 2 g$ (ดูรูปที่ 6.8) รูปที่ 6.9 แสดงศักย์แม่เหล็กบนพื้นผิว (SMP) ของหัวอ่านแบบ MR ที่มี t = 4 nm, $W = 2 0$ nm, และ $g = 1 0$ ทm และรูปที่ 6.10 แสดงรูปคอนทัวร์ของ SMP $\ddot { \textmd { 1 } }$

หมายเหตุค์ $\Psi _ { s }$ ของหัวอ่านในรูปที่ 6.9 หาได้ดังนี้ เริ่มต้นให้หาค่า $\Psi _ { s } ^ { \mathrm { h a l f } }$ จากสมการ (6.7) สำหรับ $- \tilde { g } / 2 \le x \le \tilde { g } / 2$ และ $- W / 2 \le z \le W / 2$

เพียงครึ่งส่วนของI เท่านั้น ดังนั้นถ้าสมมุติว่า I มีความสมมาตร อีกครึ่งส่วนของ ก็คือ ส่วนที่เป็นภาพสะท้อนของ $\Psi _ { s } ^ { \mathrm { h a l f } }$ ซึ่งเมื่อนำเอา Ihal และส่วนที่เป็นภาพสะท้อนของ $\Psi _ { s } ^ { \mathrm { h a l f } }$ มา ประกบกัน ก็จะได้เป็นค่า I, ซึ่งมีลักษณะเป็นเมทริกซ์ที่มีสมาชิกส่วนใหญ่ในแถวกลางเป็นค่าหนึ่ง (ซึ่งแสดงถึงค่าศักย์แม่เหล็กบนหัวอ่าน) เนื่องจากศักย์แม่เหล็กบนหัวอ่านมีค่าเท่ากับหนึ่งเสมอ ดังนั้นให้ทำสำเนาข้อมูลในแถวกลางนี้ แล้วแทรกเพิ่มเข้าไปในเมทริกช์รอบแถวกลางเดิมหลายๆ แถวตามจำนวนที่สอดคล้องกับความหนาของชินส่วน MR

ศักย์แม่เหล็กในสมการ (6.6) สามารถหาได้จากค่า รMP ในสมการ (6.7) โดยใช้อินทิกรัล เชิงตัวเลข (numerical integral) นอกจากนี้ในการจำลองชั้น SUL (soft magnetic underlayer) ของสื่อบันทึกแบบแนวตั้ง จะสมมุติว่าชั้น รบL เป็นแบบสมบูรณ์ที่มีความหนาเป็นค่าอนันต์ ถ้าให้ หัวอ่านมีภาพสมบูรณ์ (perfect image) จะได้ว่าฟังก์ชันความไวรวม (total sensitivity function) ประกอบด้วยฟังก์ชันความไวของหัวอ่านและฟังก์ชันความไวของภาพหัวอ่าน (read-head's image) เมื่อเทียบกับขอบของชั้น SUL [116] ดังนั้นฟลักซ์แม่เหล็กรวมทีใส่เข้าไปในชิ้นส่วน MR ที่พื้นผิว ของ ABS สามารถเขียนเป็นสมการคณิตศาสตร์ได้คือ

$$
\phi ( x , z ) = \frac { \mu _ { 0 } } { i } \intop \intop \intop \intop \limits _ { - \infty } ^ { \infty } \intop \limits _ { d } ^ { + \delta } \intop \Psi _ { \mathrm { t o t a l } } ( x ^ { \prime } , y ^ { \prime } , z ^ { \prime } ) \times [ \frac { \partial M _ { y } ( x ^ { \prime } - x , y ^ { \prime } , z ^ { \prime } - z ) } { \partial y ^ { \prime } } ] \} d x ^ { \prime } d y ^ { \prime } d z ^ { \prime }\tag{6.8}
$$

เมื่อ $\Psi _ { \mathrm { t o t a l } } = \Psi \big ( x ^ { \prime } , y ^ { \prime } , z ^ { \prime } \big ) + \Psi _ { \mathrm { i m a g e } } \big ( x ^ { \prime } , y ^ { \prime } , z ^ { \prime } \big )$ นอกจากนี้ถ้าสมมุติว่าไม่มีช่องว่างระหว่างชั้น SUL และสื่อบันทึก ก็จะได้ว่า $\Psi _ { \mathrm { i m a g e } } \left( x ^ { \prime } , y ^ { \prime } , z ^ { \prime } \right) = - \Psi \left( x ^ { \prime } , y = d + 2 \delta , z ^ { \prime } \right)$ ตามที่แสดงในรปที่ 6.11 และถ้าสมมุติอีกว่าสภาพความเป็นแม่เหล็กแบบแนวตัง $M _ { y }$ เป็นแบบเอกรูป (uniform) ตามแนว ความหนาของไอแลนด์ดั้งแสดงในรูปที่ 6.12 (นั้นคืออนุพันธ์ของ $M _ { y }$ เมื่อเทียบกับ $y$ ก็คือฟังก์ชัน 3ื่ป น อิมพัลส์สองฟังก์ชัน) ดังนันผลตอบสนองสัญญาณพัลส์หรือสัญญาณอ่านกลับที่ได้จากหัวอ่านที่มี จุดศูนย์กลางอยู่ที่พิกัด (0, 0) ที่เป็นผลมาจากไอแลนด์ที่มีจุดศูนย์กลางอยู่ที่พิกัด (x, y) สามารถ เขียนเป็นสมการคณิตศาสตร์ได้คือ

$$
V \big ( x , z \big ) = C \int _ { - \infty } ^ { \infty } \Big \{ M \big ( x ^ { \prime } - x , y ^ { \prime } , z ^ { \prime } - z \big ) \Big [ \Psi \big ( x ^ { \prime } , y ^ { \prime } = d , z ^ { \prime } \big ) - \Psi \big ( x ^ { \prime } , y ^ { \prime } = d + 2 \delta , z ^ { \prime } \big ) \Big ] \Big \} d x ^ { \prime }\tag{6.9}
$$

เมื่อ

$$
M \left( x , z \right) = \left\{ \begin{array} { l l } { { M \ ( \mathrm { m e d i a \ m a g n e t i z a t i o n } ) , } } & { { x , z \in \ \mathrm { i s l a n d } } } \\ { { 0 , } } & { { \mathrm { e l s e } } } \end{array} \right.\tag{6.10}
$$

สมการ (6.10) บอกให้ทราบว่าสภาพความเป็นแม่เหล็กแบบแนวตั้ง $M _ { y }$ มีค่าเท่ากับค่าคงตัว M เฉพาะบริเวณที่เป็นไอแลนด์ (นอกนั้นมีค่าเท่ากับศูนย์) เพราะฉะนั้นสมการ (6.9) สามารถจัดรูป ใหม่ได้คือ

$$
V { \big ( } x , z { \big ) } = \tilde { C } \int _ { x - a / 2 } ^ { x + a / 2 z + a / 2 } \int \displaylimits _ { z - a / 2 } ^ { } \left\{ \Psi { \big ( } x ^ { \prime } , d , z ^ { \prime } { \big ) } - \Psi { \big ( } x ^ { \prime } , d + 2 \delta , z ^ { \prime } { \big ) } \right\} d x ^ { \prime } d z ^ { \prime }\tag{6.11}
$$

เมื่อ $\tilde { C } = C M$ คือค่าคงตัว สมการ (6.11) ทำให้สามารถสร้างผลตอบสนองสัญญาณพัลส์แบบ สองมิติ (2D pulse response) สำหรับรูปทรงของหัวอ่านและไอแลนด์เชิงแม่เหล็กลักษณะใดๆ ได้โดยง่าย รูปที่ 6.13 แสดงผลตอบสนองสัญญาณพัลส์แบบสองมิติสำหรับไอแลนด์ที่มีความยาว 11 นาโนเมตร (a = 11 nm), ความสูง 10 นาโนเมตร (δ = 10 nm), และระยะบิน 10 นาโนเมตร (d = 10 m) โดยใช้หัวอ่านแบบ MR ที่มีพารามิเตอร์ตามรูปที่ 6.9 และรูปที่ 6.14 แสดงรูป คอนทัวร์ของสัญญาณพัลส์แบบสองมิติในรูปที่ 6.13

ภาคตัดขวางของสัญญาณพัลส์แบบสองมิติในแนวตามแทร็ก (z = 0) เรียกว่า "ผลตอบ สนองสัญญาณพัลส์ตามแทร็ก (along-track pulse response)" และภาคตัดขวางของสัญญาณพัลส์ แบบสองมิติในแนวขวางแทร็ก (x = 0) จะเรียกว่า "โพรไฟล์ขวางแทร็ก (cross-track profile)" รูปที่ 6.15 แสดงผลตอบสนองสัญญาณพัลส์ตามแทร็กและโพรไฟล์ขวางแทร็กของสัญญาณพัลส์ แบบสองมิติในรูปที่ 6.13 จากรูปจะเห็นได้ว่าโพรไฟล์ขวางแทร็กมีความกว้างมากกว่าผลตอบสนอง สัญญาณพัลส์ตามแทร็ก ซึ่งหมายความว่าระบบ BPMR37 จะเผชิญกับการแทรกสอดระหว่างแทร็ก (ITI) ที่มีความรุนแรงมากกว่าการแทรกสอดระหว่างสัญลักษณ์ (เIร1) ถ้ากำหนดให้ $\mathrm { P W } _ { 5 0 }$ คือ ความกว้างของสัญญาณพัลส์วัด ณ ตำแหน่งที่สัญญาณพัลส์มีความสูงเป็นครึ่งหนึ่งของความสูง สูงสุด ดังนั้นรุปที่6.15 บอกให้ทราบว่า $\mathrm { P W } _ { 5 0 }$ ในแนวตามแทร็กมีค่าเท่ากับ 22.1 นาโนเมตร และ $\mathrm { P W } _ { 5 0 }$ ในแนวขวางแทร็กมีค่าเท่ากับ 30 นาโนเมตร

## 6.3.2 ผลกระทบที่เกิดจากไอแลนด์และรูปทรงของหัวอ่าน

จากที่กล่าวมาในตอนต้นพบว่าผลตอบสนองสัญญาณพัลส์แบบสองมิติในระบบ BPMR จะขึ้นอยู กับลักษณะของไอแลนด์เชิงแม่เหล็ก (เช่น รูปร่างและขนาดของไอแลนด์) และรูปทรงของหัวอ่าน (เช่น ความกว้าง W และช่องว่าง $g )$ ดังนั้นหัวข้อนี้จะศึกษาผลกระทบของลักษณะของไอแลนด์ และรูปทรงของหัวอ่านที่มีต่อผลตอบสนองสัญญาณพัลส์แบบสองมิตินี้ โดยจะพิจารณาเฉพาะ ไอแลนด์แบบสี่เหลี่ยมจัตรัสและหัวอ่านแบบ MR เท่านั้น ซึ่งการศึกษานี้ช่วยทำให้ผู้อ่านเข้าใจถึง การเลือกค่าพารามิเตอร์ต่างๆ ที่ใช้ในระบบการบันทึกข้อมูลแบบ BPMR

รูปที่ 6.16 แสดงแอมพลิจูดที่ถูกทำให้เป็นบรรทัดฐาน (normalized amplitนde) ของ สัญญาณพัลส์ตามแทร็ก (ของแทร็กหลัก) ที่เป็นฟังก์ชันของความยาวของไอแลนด์ (เมื่อเทียบกับ แอมพลิจูดของสัญญาณพัลส์เมื่อความยาวของไอแลนด์เท่ากับ 13 ทm) และรูปที่ 6.17 แสดงค่า $\mathrm { P W } _ { 5 0 }$ ในแนวตามแทร็ก (along-track) และในแนวขวางแทร็ก (cross-track) มีหน่วยเป็นนาโน เมตร ที่เป็นฟังก์ชันของความยาวของไอแลนด์ โดยสัญญาณพัลส์แบบสองมิติที่ใช้ในรูปที่ 6.16 และ 6.17 ได้มาจากไอแลนด์แบบสี่เหลี่ยมจัตุรัสที่มีความสูงของไอแลนด์ $\delta = 1 0$ nm และระยะ บินของหัวอ่าน d = 10 nm และใช้หัวอ่านแบบ MR ที่มีความหนา t = 4 nm, ความกว้าง $W =$ 20 ทm, และความกว้างของช่องว่างระหว่างฉนวนและชินส่วน MR เท่ากับ $g = 1 0$ nm

รูปที่ 6.16 แอมพลิจูดที่ถูกทำให้เป็นบรรทัดฐานของสัญญาณพัลส์ตามแทร็กที่เป็นฟังก์ชันของความยาวของ ไอแลนด์ (เมื่อเทียบกับแอมพลิจูดของสัญญาณพัลส์เมื่อ a = 13 nm)

จากรูปที่ 6.16 พบว่าแอมพลิจูดของสัญญาณพัลส์แบบสองมิติมีค่ามากขึ้น เมื่อความยาว ของไอแลนด์เพิ่มขึ้น ซึ่งหมายความว่าระบบ BPMR ที่ใช้ไอแลนด์ขนาดใหญ่จะมีผลทำให้กำลังของ สัญญาณอ่านกลับมีค่าสูงขึ้น (ซึ่งช่วยทำให้ง่ายต่อการถอดรหัสข้อมูล) อย่างไรก็ตามรูปที่ 6.17 บอก ให้ทราบว่าเมื่อไอแลนด์มีขนาดใหญ่ขึ้น ค่า $\mathrm { P W } _ { 5 0 }$ ในแนวตามแทร็กและในแนวขวางแทร็กก็จะมี ค่ามากขึ้น ซึ่งหมายความว่าระบบจะเผชิญกับปัญหาเรื่องการแทรกสอดระหว่างสัญลักษณ์ (ISI) และการแทรกสอดระหว่างแทร็ก (IT1) มากขึนด้วย ดังนันนักออกแบบระบบจะต้องประนีประนอม ระหว่างกำลังของสัญถาณอ่านกลับที่ต้องการและระดับความรนแรงของ ISI และ ITI ที่ยอมรับได้ ตัวอย่างเช่น ถ้าต้องการให้ระบบ BPMR มีความหนาแน่นเชิงพื้นที่เท่ากับ 1 Tb/in2 ก็ควรมีคาบ เวลาของบิตและระยะแทร็กน้อยกว่า 25 ทm [108] และควรใช้ความยาวของไอแลนด์เท่ากับ 11 nm หรือ 13 ทm เพื่อให้มีระยะห่างระหว่างไอแลนด์เพียงพอ (มี IS1 และ ITI น้อย) และได้กำลังของ สัญญาณอ่านกลับที่เพียงพอสำหรับการถอดรหัสข้อมูล

![](images/chapter_6/ec0a30906cb8d5e3a79a3e6607ef9b72889e5829e63e1a5b434b456cf67b7747.jpg)  
รูปที่ 6.18 $\mathrm { P W } _ { 5 0 }$ ที่เป็นฟังก์ชันของความกว้างของชิ้นส่วน MR (พารามิเตอร์ W)

รูปที่ 6.18 แสดงค่า $\mathrm { P W } _ { 5 0 }$ ในแนวตามแทร็กและในแนวขวางแทร็กที่เป็นฟังก์ชันของ ความกว้างของชึ้นส่วน MR (นั่นคือพารามิเตอร์ W) และรูปที่ 6.19 แสดงแอมพลิจูดที่ถูกทำให้ เป็นบรรทัดฐานของสัญญาณพัลส์ตามแทร็กที่เป็นฟังก์ชันของความกว้างของชิ้นส่วน MR (เทียบ กับแอมพลิจูดของสัญญาณพัลส์เมื่อความกว้างของชิ้นส่วน MR เท่ากับ 20 ทm) โดยสัญญาณ พัลส์แบบสองมิติที่ใช้ในรูปที่ 6.18 และ 6.19 ได้มาจากไอแลนด์แบบสี่เหลี่ยมจัตุรัสที่มีความยาว ของไอแลนด์ $a = 1 3$ nm, ความสูงของไอแลนด์ 8 = 10 nm, และระยะบินของหัวอ่าน d = 10 ทm และใช้หัวอ่านแบบ MR ที่มีความหนา t = 4 nm และความกว้างของช่องว่างระหว่างฉนวน และชิ้นส่วน MR เท่ากับ g = 10 nm

รูปที่ 6.18 แสดงให้เห็นว่าเมื่อความกว้างของชึ้นส่วน MR มีค่าเพิ่มขึ้น ค่า $\mathrm { P W } _ { 5 0 }$ ในแนว ตามแทร็กจะมีการเปลี่ยนแปลงอย่างช้าๆ ในขณะที่ค่า $\mathrm { P W } _ { 5 0 }$ ในแนวขวางแทร็กจะมีการเปลี่ยนแปลง อย่างรวดเร็ว ซึ่งหมายความว่าความกว้างของชึ้นส่วน MR มีผลกระทบต่อความรุนแรงของ ITI มากกว่า ISI นอกจากนี้รูปที่ 6.19 บอกให้ทราบว่าเมื่อความกว้างของชึ้นส่วน MR มีค่าเพิ่มขึ้น แอมพลิจูดของสัญญาณพัลส์แบบสองมิติจะมีค่าเพิ่มขึ้นอย่างช้าๆ และเริ่มลู่เข้าสู่ค่าคงตัว ดังนั้นถ้า ต้องการให้ระบบมี ITI น้อย ก็ควรใช้หัวอ่านแบบ MR ที่มีความกว้าง W น้อย อย่างไรก็ตามการ สร้างหัวอ่านที่มีความกว้างน้อยยังคงเป็นสิ่งที่ท้าทาย (สร้างยากและมีราคาแพง) ดังนั้นในยุคแรก ระบบ BPMR จะใช้ความกว้างของชิ้นส่วน MR ประมาณ 15 - 20 nm [108]

![](images/chapter_6/fa9bb0d2e2d59a2cf8e8c5687334c700da56cb095c2f4581e6b0c5e6101ea932.jpg)  
รูปที่ 6.19 แอมพลิจูดที่ถูกทำให้เป็นบรรทัดฐานของสัญญาณพัลส์ตามแทร็กที่เป็นฟังก์ชันของความกว้างของ ชึ้นส่วน MR (เมื่อเทียบกับแอมพลิจูดของสัญญาณพัลส์เมื่อ W = 20 nm)

ในทำนองเดียวกันรูปที่ 6.20 แสดงค่า $\mathrm { P W } _ { 5 0 }$ ในแนวตามแทร็กและในแนวขวางแทร็กที่ เป็นฟังก์ชันของความกว้างของช่องว่างระหว่างฉนวนและชิ้นส่วน MR (พารามิเตอร์ ) และรูปที 6.21 แสดงแอมพลิจดที่ถูกทำให้เป็นบรรทัดฐานของสัญถาณพัลส์ตามแทร็กที่เป็นฟังก์ชันของ ความกว้างของช่องว่างระหว่างฉนวนและชึ้นส่วน MR (เทียบกับแอมพลิจูดของสัญญาณพัลส์เมื่อ g = 10 ทm) โดยสัญญาณพัลส์แบบสองมิติที่ใช้ในรูปที่ 6.20 และ 6.21 ได้มาจากไอแลนด์แบบ สีเหลียมจัตุรัสที่มีความยาว a = 13 ทm, ความสูง 8 = 10 ทm, และระยะบินของหัวอ่าน d = 10 ทm และใช้หัวอ่านแบบ MR ที่มีความหนา t = 4 nm และความกว้างของชิ้นส่วน MR เท่ากับ W = 20 nm

เพิ่มขึ้น ค่า $\mathrm { P W } _ { 5 0 }$ ในแนวตามแทร็กและในแนวขวางแทร็กก็จะมีค่ามากขึ้นด้วย ซึ่งหมายความว่า ค่า ITI และ ISI สามารถถูกควบคุมได้ โดยการเปลี่ยนแปลงความกว้างของช่องว่างระหว่างฉนวน และชิ้นส่วน MR นอกจากนี้รูปที่ 6.21 บอกให้ทราบว่าแอมพลิจูดของสัญญาณพัลส์แบบสองมิติ จะมีค่าเพิ่มขึ้น เมื่อความกว้างของช่องว่างระหว่างฉนวนและชึ้นส่วน MR มีค่ามากขึ้น (แต่ไม่ รวดเร็วเหมือนการเปลี่ยนแปลงของค่า $\mathrm { P W } _ { 5 0 }$ ในรูปที่ 6.20) อย่างไรก็ตามในทางปฏิบัติหัวอ่าน แบบ MR ที่ใช้ในระบบ BPMR จะมีความกว้างของช่องว่างระหว่างฉนวนและชึ้นส่วน MR ประมาณ 6 – 10 nm [108]

## 6.4 แบบจำลองของสัญญาณอ่านกลับในระบบ BPMR

โดยทั่วไปสัญญาณอ่านกลับในระบบการบันทึกข้อมูลเชิงแม่เหล็กจะได้มาจากการซ้อนทับเชิงเส้น (linear รนperpoรitoก)ของผลตอบสนองสัญญาณพัลส์ในแนวตามแทร็ก อย่างไรก็ตามสำหรับ ระบบ BPMR ที่มีความจุข้อมูลสูง ระยะห่างระหว่างไอแลนด์จะมีค่าน้อยมาก ดังนั้นผลกระทบที่ เกิดจากแทร็กข้างเคียง (adjacent track) จึงเป็นสิ่งที่หลีกเลี่ยงไม่ได้ จากรูปที่ 6.14 และ 6.15 จะ เห็นได้ว่าโพรไฟล์ขวางแทร็ก (croรร-track profile) มีความกว้างมากกว่าผลตอบสนองสัญญาณ e   
พัลส์ตามแทร็ก (along-track pulse) เพราะฉะนั้นสัญญาณอ่านกลับของระบบ BPMR ที่รวม ผลกระทบที่เกิดจากแทร็กข้างเคียงจะได้มาจากการซ้อนทับเชิงเส้นของผลตอบสนองสัญญาณพัลส์ แบบสองมิติ (เทียบกับข้อมูลอินพุตแบบไบนารีแบบสองมิติ)

เนื่องจากตำแหน่งของไอแลนด์ (หรือบิตข้อมูล) ในระบบ BPMR ได้ถูกกำหนดไว้แล้ว ดังนั้นสัญญาณอ่านกลับจึงสามารถถูกแทนด้วยตำแหน่งของบิตข้อมูล (แทนที่จะเป็นดรรชนีเวลา) ให้พิจารณาไอแลนด์เชิงแม่เหล็กแบบจัตุรัสตามรูปที่ 6.6 (ก) จะได้ว่าสัญญาณอ่านกลับที่ต่อเนื่อง ทางเวลาสามารถเขียนเป็นสมการคณิตศาสตร์ได้คือ

$$

$$

## 6.4.1 ค่าประมาณของสัญญาณพัลส์ BPMR

เนืองจากผลตอบสนองสัญญาณพัลส์แบบสองมิติ $P ( z , \ x )$ หรือเรียกสันๆ ว่าสัญญาณพัลส์ BPMR ที่มีความละเอียดสูง (เพื่อให้สัญญาณอ่านกลับมีความถูกต้องและเหมือนกับสัญญาณจริงที่ได้จาก หัวอ่านให้มากสุด) สร้างได้ค่อนข้างยาก ดังนั้นเพื่อให้ง่ายต่อการวิเคราะห์ระบบ BMR งานวิจัย ใน [108] จึงได้ประมาณค่า (approximation) สัญญาณพัลส์ BPMR ให้เป็นสัญญาณพัลส์เกาส์ เซียนแบบสองมิติซึ่งเขียนเป็นสมการคณิตศาสตร์ได้คือ

$$
P \left( z , x \right) = A \exp \left\{ - \frac { 1 } { 2 } \Biggl ( \frac { x ^ { 2 } } { W _ { x } ^ { 2 } } + \frac { z ^ { 2 } } { W _ { z } ^ { 2 } } \Biggr ) \right\}\tag{6.14}
$$

เมื่อ A คือแอมพลิจูดสูงสุดมีค่าเท่ากับ 1, $W _ { x } = \mathrm { P W } _ { 5 0 } ^ { \mathrm { a l o n g } } / 2 . 3 5 4 8$ และ $W _ { z } = \mathrm { P W } _ { 5 0 } ^ { \mathrm { c r o s s } }$ /2.3548 โดย $\mathrm { P W _ { 5 0 } ^ { a l o n g } }$ คือค่า $\mathrm { P W } _ { 5 0 }$ ของสัญญาณพัลส์ตามแทร็ก, $\mathrm { P W } _ { 5 0 } ^ { \mathrm { c r o s s } }$ คือค่า $\mathrm { P W } _ { 5 0 }$ ของโพรไฟล์ขวาง แทร็ก, และค่า 2.3548 คือค่าคงตัวของความสัมพันธ์ระหว่างค่า $\mathrm { P W } _ { 5 0 }$ และส่วนเบี่ยงเบนมาตรฐาน ของฟังก์ชันเกาส์เซียน [108]

รูปที่ 6.23 แสดงผลตอบสนองสัญญาณพัลส์เกาส์เซียนแบบสองมิติตามสมการ (6.14) ที่มี $\mathrm { P W } _ { 5 0 } ^ { \mathrm { a l o n g } } = 2 2 . 1$ nmและ $\mathrm { P W } _ { 5 0 } ^ { \mathrm { c r o s s } } = 3 0$ nm ซึ่งมีรูปร่างคล้ายกับสัญญาณพัลส์ BPMR ใน รูปที่6.13 (ซึ่งมี $\mathrm { P W } _ { 5 0 } ^ { \mathrm { a l o n g } } = 2 2 . 1$ nm และ $\mathrm { P W } _ { 5 0 } ^ { \mathrm { c r o s s } } = 3 0$ ทm) นอกจากนี้รูปที่ 6.24 เปรียบเทียบ สัญญาณพัลส์ตามแทร็กและโพรไฟล์ขวางแทร็กของสัญญาณพัลส์ BPMR ในรูปที่ 6.13 และของ สัญญาณพัลส์เกาส์เซียนแบบสองมิติตามสมการ (6.14) (แสดงด้วยเส้นปะ) จากรูปที่ 6.24 พบว่า สัญญาณพัลส์ตามแทร็กและโพรไฟล์ขวางแทร็กของสัญญาณพัลส์ BPMR และของสัญญาณพัลส์ เกาสเซียนแบบสองมิติมีค่าใกล้เคียงกัน ดังนั้นสรุปได้ว่าสัญญาณพัลส์เกาส์เซียนแบบสองมิติใน สมการ (6.14) สามารถนำมาใช้ประมาณค่าสัญญาณพัลส์ BPMR ในรูปที่ 6.13 ได้ เพื่อให้ง่าย ต่อการวิเคราะห์ระบบ BPMR

## 6.4.2 ค่าประมาณของสัญญาณพัลส์ BPMR ที่มีสัญญาณรบกวนสื่อบันทึก

จากที่กล่าวมาในตอนต้น ช่องสัญญาณ BPMR สามารถลดปัญหาเรื่องสัญญาณรบกวนการเปลี่ยน สถานะ (transition noise) และการเลื่อนบิตแบบไม่เป็นเชิงเส้น (nonlinear bit รhift) อย่างไรก็ตาม สัญญาณรบกวนสื่อบันทึก (media noise) เป็นสิ่งที่หลีกเลี่ยงไม่ได้ในช่องสัญญาณ BPMR ซึ่งมี ผลให้สมรรถนะรวมของระบบลดลงมาก โดยต้นกำเนิดของสัญญาณรบกวนสือบันทึกมาจาก 5 สาเหตุหลักดังนี้ [108]

1) ความผันผวนของตำแหน่ง (location fใuctนation) ของแต่ละไอแลนด์ กล่าวคือเนื่องจากแต่ละ ไอแลนด์มีระยะห่างน้อยกว่า 25 นาโนเมตร จึงเป็นการยากที่ทำให้ทุกไอแลนด์มีระยะห่างเท่ากัน อย่างสม่ำเสมอ

2) ความผันผวนของขนาด (size fuctนation) ของแต่ละไอแลนด์

3) ความผันผวนของความสูง (height flนctนation) ของแต่ละไอแลนด์

4) ความผันผวนของรูปร่าง (shape fluctuation) ของแต่ละไอแลนด์

5) ความผันผวนของสภาพความเป็นแม่เหล็กอิ่มตัว (saturation magnetization)เนื่องจากแต่ละ ไอแลนด์มีขนาดไม่แน่นอนจึงทำให้สภาพความเป็นแม่เหล็กของแต่ละไอแลนด์มีความอิ่มตัว ของแม่เหล็กที่ไม่เท่ากัน

อย่างไรก็ตามสัญญาณรบกวนสื่อบันทึกที่พบมากในระบบ BPMR [108] เกิดจากความ ผันผวนของตำแหน่งและขนาดของแต่ละไอแลนด์ดั้งแสดงในรูปที่ 6.25 ดังนั้นค่าประมาณของ สัญญาณพัลส์ BPMR ที่มีสัญญาณรบกวนสือบันทึกที่เกิดจากความผันผวนของตำแหน่งและขนาด ของไอแลนด์สามารถเขียนเป็นสมการคณิตศาสตร์ได้คือ [108]

$$
P \left( z , x \right) = \left( A + \Delta _ { A } \right) \exp \left\{ - \frac { 1 } { 2 } \left[ \left( \frac { x + \Delta _ { x } } { c \left( W _ { x } + \Delta _ { W _ { x } } \right) } \right) ^ { 2 } + \left( \frac { z + \Delta _ { z } } { c \left( W _ { z } + \Delta _ { W _ { z } } \right) } \right) ^ { 2 } \right] \right\}\tag{6.15}
$$

## 6.4.3 แบบจำลองช่องสัญญาณ BPMR ที่ไม่ต่อเนื่องทางเวลาแบบสมมูล

เมื่อสัญญาณอ่านกลับ $s ( x )$ ในสมการ (6.12) ถูกทำการชักตัวอย่าง ณ เวลาที่เป็นระยะห่างระหว่าง ไอแลนด์ ก็จะได้สัญญาณอ่านกลับที่ไม่ต่อเนื่องทางเวลาซึ่งเขียนเป็นสมการคณิตศาสตร์ได้คือ

$$
r _ { k } = s ( x ) { \Big | } _ { x = k T _ { x } } = \sum _ { m = - M } ^ { M } \sum _ { n = - N } ^ { N } a _ { m , n } H \bigl ( - m , k - n \bigr ) + n _ { k }\tag{6.16}
$$

เมื่อ $n _ { k }$ คือสัญญาณรบกวนเกาส์สีขาวแบบบวก (AWGN: additive white Gaussian noise) และ H(m, ท) คือช่องสัญญาณแบบสองมิติที่ไม่ต่อเนื่องทางเวลาซึ่งได้จากการชักตัวอย่างสัญญาณพัลส์ $P ( z , \ x )$ ณ เวลาที่เป็นจำนวนเท่าของคาบเวลาของบิตและระยะแทร็ก นั้นคือ

$$
H { \left( m , n \right) } = P { \left( m T _ { z } , n T _ { x } \right) }\tag{6.17}
$$

สำหรับ $- M \leq m < M$ และ $- N \leq n < N$ รูปที่ 6.27 แสดงแผนภาพบล็อกของช่องสัญญาณ BPMR ที่ไม่ต่อเนื่องทางเวลาแบบสมมูล โดยที่ $a _ { 0 , k }$ สอดคล้องกับลำดับข้อมูลอินพุตของแทร็กหลัก ที่จะถูกอ่านและถูกถอดรหัสข้อมูลด้วยวงจรภาครับ นอกจากนี้การคำนวณหาค่าสัญญาณอ่านกลับ $r _ { k }$ ในสมการ (6.16) จะต้องอาศัยการทำคอนโวลูชันแบบสองมิติ (2D convoในtion) ซึ่งค่อนข้าง ซับซ้อน [118]

เนื่องจากช่องสัญญาณ $H ( m , n )$ ในสมการ (6.17) สามารถเขียนให้อยูในรูปเมทริกซ์ H [108, 109 จะพิจารณาใช้ข้อมูลเพียง 3 คาบเวลาของบิต และ 3 แทร็ก ตามลำดับ เพราะระบบ BPMR ที่ความจุข้อมูลสูงๆ (มากกว่า $1 \ \mathrm { T b } / \mathrm { i n } ^ { 2 } )$ ผลกระทบที่เกิดจากบิตและแทร็กข้างเคียงอื่นๆ จะมีค่าน้อยมาก (สามารถละทิ้งได้) นอกจากนี้เพื่อให้ง่ายต่อการจำลองระบบและการสร้างสัญญาณ อ่านกลับทีไม่ต่อเนื่องทางเวลาของระบบ BPMR Karakนlak et al. [112] ได้นำเสนอแบบจำลอง ช่องสัญญาณ BPMR ที่ไม่ต่อเนื่องทางเวลาตามรูปที่ 6.28 โดยที่ช่องสัญญาณ BPMR ซึ่งแทนด้วย เมทริกซ์ H มีค่าเท่ากับ

![](images/chapter_6/d1ce00a39be9d0769f3a03a9d812cbca05298591a0e194b80a6bde23cbe70639.jpg)  
รูปที่ 6.28 แบบจำลองช่องสัญญาณ BPMR ที่ไม่ต่อเนื่องทางเวลา

$$
\begin{array} { r l } & { \mathbf { H } = \left[ \begin{array} { l } { H _ { - 1 } \left( D \right) } \\ { H _ { 0 } \left( D \right) } \\ { H _ { 1 } \left( D \right) } \end{array} \right] = \left[ \begin{array} { l } { h _ { - 1 , - 1 } D ^ { - 1 } + h _ { - 1 , 0 } + h _ { - 1 , 1 } D } \\ { h _ { 0 , - 1 } D ^ { - 1 } + h _ { 0 , 0 } + h _ { 0 , 1 } D } \\ { h _ { 1 , - 1 } D ^ { - 1 } + h _ { 1 , 0 } + h _ { 1 , 1 } D } \end{array} \right] } \\ & { = \left[ \begin{array} { l l l } { h _ { - 1 , - 1 } } & { h _ { - 1 , 0 } } & { h _ { - 1 , 1 } } \\ { h _ { 0 , - 1 } } & { h _ { 0 , 0 } } & { h _ { 0 , 1 } } \\ { h _ { 1 , - 1 } } & { h _ { 1 , 0 } } & { h _ { 1 , 1 } } \end{array} \right] } \end{array}\tag{6.18}
$$

และสัญญาณอ่านกลับ $r _ { k }$ สามารถเขียนให้อยู่ในรูปการทำคอนโวลูชันแบบหนึ่งมิติได้คือ

$$
r _ { k } = \left( a _ { 0 , k } * h _ { 0 , k } \right) + \left( a _ { - 1 , k } * h _ { - 1 , k } \right) + \left( a _ { 1 , k } * h _ { 1 , k } \right) + n _ { k }\tag{6.19}
$$

เมื่อ \* คือตัวดำเนินการคอนโวลูชัน (convolนtion operator) จากนั้นสัญญาณอ่านกลับ $r _ { k }$ ก็จะ ถูกส่งเข้าไปในวงจรตรวจหา PRML (partial-response maximum-likelihood) [10, 27] เพื่อหา ค่าประมาณของลำดับข้อมูลของแทร็กหลัก $( \hat { a } _ { 0 , k } )$ เท่านั้น

วงจรตรวจหา PRML เป็นการทำงานร่วมกันระหว่างอีควอไลเซอร์และวงจรตรวจหาวีเทอร์บิ [10, 13] โดยอีควอไลเซอร์จะทำหน้าที่ปรับรูปร่างของสัญญาณให้เป็นไปตาม “ทาร์เก็ต (target)" ที่ต้องการ และวงจรตรวจหาวีเทอร์บิจะทำหน้าที่ถอดรหัสข้อมูลโดยใช้วงจรตรวจหาวีเทอร์บิที่สร้าง

![](images/chapter_6/e53708ffbdeea57aefe375f3f6eeae5c2ba9a1e4b439687620777ce9404681c7.jpg)  
รูปที่ 6.29 ลักษณะของแทร็กมิสเรจิสเตรชัน (TMR) ในระบบ BPMR

จากทาร์เก็ต4ที่กำหนดไว้ เพราะฉะนั้นจะเห็นได้ว่าวงจรตรวจหา PRML ที่ใช้ในการถอดรหัสข้อมูล ของระบบการบันทึกข้อมูลเชิงแม่เหล็กแบบแนวนอนและแบบแนวตั้ง [10] ยังคงสามารถนำมาใช้ ถอดรหัสข้อมูลของระบบ BPMR ได้อย่างมีประสิทธิภาพเช่นกัน [108, 109]

## 6.4.4 แทร็กมิสเรจิสเตรชัน

รูปที่ 6.7 แสดงตำแหน่งที่เหมาะสุดของหัวอ่านแบบ MR ที่อยู่บนสื่อบันทึก กล่าวคือจุดศูนย์กลาง ของหัวอ่านควรอยู่ณ จุดกึ่งกลางของแทร็กหลักและเคลื่อนที่ขนานไปกับแนวตามแทร็ก เพื่อให้ สัญญาณอ่านกลับที่ด้านขาออกของหัวอ่านมีคุณภาพดีสุด อย่างไรก็ตามในทางปฏิบัติหัวอ่านมี โอกาสเคลื่อนที่เลื่อนออกไปจากตำแหน่งที่ควรจะเป็น โดยถ้าจุดศูนย์กลางของหัวอ่านจะเคลื่อนที่ ไปในแนวขวางแทร็กทางด้านบน (หรือด้านล่าง) ตามที่แสดงในรูปที่ 6.29 จะเรียกปรากฎการณ์นี้ ว่า "แทร็กมิสเรจิสเตรชัน" (TMR: track mis-registration) หรือออฟเซ็ตของหัวอ่าน (read head offset) ซึ่งถูกกำหนดโดย [119]

$$
\mathrm { T M R } = \frac { \Delta _ { T } } { T _ { z } } { \times } 1 0 0 \tag{6.20}
$$

มีหน่วยเป็นเปอร์เซ็นต์ เมื่อ $\Delta _ { T }$ (track offset) คือระยะที่หัวอ่านเคลื่อนที่ออกไปจากแทร็กหลัก ในแนวขวางแทร็ก และ $T _ { z }$ คือระยะแทร็ก (track pitch) ดังนั้นสัญญาณพัลส์ BPMR แบบที่ไม่ ต่อเนื่องทางเวลาที่มีผลกระทบของแทร็กมิสเรจิสเตรชันมีค่าเท่ากับ

![](images/chapter_6/7c1604afcb4d0fa85ba3c49ed41bca9988884b419515dd076bd9fe0c7af10a14.jpg)  
รูปที่ 6.30 สัญญาณพัลส์ตามแทร็กและสัญญาณพัลส์ข้ามแทร็กที่มีผลกระทบของ TMR

$$
H { \big ( } m , n { \big ) } = P { \big ( } m T _ { z } + \Delta _ { T } , n T _ { x } { \big ) }\tag{6.21}
$$

ในทางปฏิบัติ TMR มีผลกระทบต่อสมรรถนะของระบบ BPMR ค่อนข้างมาก เพราะมีผลให้ สัญญาณอ่านกลับที่ได้มีรูปร่างผิดเพี้ยนไปจากที่ควรจะเป็น รูปที่ 6.30 แสดงสัญญาณพัลส์ตาม แทร็ก41 (along-track pulse) และสัญญาณพัลส์ข้ามแทร็ก42 (cross-track pulse) ที่มีผลกระทบ ของ TMR โดยอาศัยสมการ (6.14) เมื่อ $\mathrm { P W } _ { 5 0 } ^ { \mathrm { a l o n g } } = 2 2 . 1$ nmและ $\mathrm { P W } _ { 5 0 } ^ { \mathrm { c r o s s } } = 3 0$ nm และสมมุติ ว่า $T _ { x } = \ T _ { z } = 2 2$ ทm จากรูปพบว่าเมื่อมี TMR เกิดขึ้นในระบบจะส่งผลให้สัญญาณพัลส์ตาม แทร็กและสัญญาณพัลส์ข้ามแทร็กมีแอมพลิจูดเปลี่ยนแปลงไปจากเดิม ซึ่งทำให้ยากต่อการถอดรหัส ข้อมูลที่วงจรภาครับ รูปที่ 6.31 แสดงตัวอย่างสมรรถนะของระบบ BPMR ที่เผชิญกับแทร็กมิส เรจิสเตรชันที่มีระดับความรุนแรงต่างๆ จากรูปพบว่าเมื่อแทร็กมิสเรจิสเตรชันมีความรุนแรงมากขึ้น ก็จะทำให้ระบบมีสมรรถนะลดลง ดังนั้นระบบ BPMR ควรมีกระบวนการป้องกันและแก้ไขปัญหา เรื่องแทร็กมิสเรจิสเตรชัน เพื่อให้ระบบ BPMR มีสมรรถนะสูงสุด

![](images/chapter_6/d52cae434c81694350762ea8c11750138a7afbb52db3af8039ef67816fb0cb34.jpg)  
รูปที่ 6.31 ตัวอย่างสมรรถนะของระบบ BPMR ณ ระดับความรุนแรงต่างๆ ของแทร็กมิสเรจิสเตรชัน (TMR)

## 6.5 สรุปท้ายบท

เทคโนโลยีการบันทึกข้อมูลเชิงแม่เหล็กแบบแนวตั้ง (perpendicular magnetic recording) ซึ่งใช้ ในฮาร์ดิสก์ไดรฟ์ปัจุบัน43กำลังเข้าใกล้ขีดจำกัดซูเปอร์พาราแมกเนติก ดังแสดงในรูปที่ 6.32 ดังนั้นเทคโนโลยีอื่นๆ ที่สามารถเพิ่มความหนาแน่นเชิงพื้นที่ (areaลl deกรity) ได้มากกว่า 1 Tb/in2 จึงเป็นสิ่งจำเป็นอย่างยิ่ง ซึ่งได้แก่ เทคโนโลยี BPMR, เทคโนโลยี HAMR, เทคโนโลยี MAMR (microwave assisted magnetic recording), เทคโนโลยี swR (shingled write recording), และ เทคโนโลยี TDMR โดยแต่ละแบบมีความท้าทายในการออกแบบเพื่อใช้งานจริงดังนี้ [117]

1)เทคโนโลยี BPMR ยังคงมีปัญหาเกี่ยวกับการสร้างสื่อบันทึกที่มีความเรียบ (planarization) ในระดับนาโนเมตร, การออกแบบช่องสัญญาณอ่านใหม่ (new read chaททel), และการเข้า จังหวะการเขียน (write synchronization)

2) เทคโนโลยี HAMR ยังคงมีปัญหาเกี่ยวกับการสร้างหัวเขียนที่มีเลเซอร์ (aser) เพื่อให้ความร้อน กับสื่อบันทึก, กระบวนการทำให้สือบันทึกเย็นตัวลงอย่างรวดเร็ว, และตำแหน่งของการติดตั้ง เลเซอร์ที่มีผลกระทบต่อการเขียน

3) เทคโนโลยี MAMR ยังคงมีปัญหาเกี่ยวกับเครื่องกำเนิดความถี่ของสัญญาณไมโครเวฟ

![](images/chapter_6/b48aea51f4c2896c0c7e10a6863889cef7d170d508567106105cbea0572031c0.jpg)  
รูปที่ 6.32 แนวโน้มความจุข้อมูลของฮาร์ดดิสก์ไดรฟ์ [117]

4) เทคโนโลยี รพR ซึ่งใช้ร่วมงานกับเทคโนโลยี TDMR ยังคงมีปัญหาเกี่ยวกับการประมวลผล สัญญาณแบบสองมิติ (2D signal processรing) ขั้นสูงในช่องสัญญาณอ่าน

อย่างไรก็ตามบทนี้ได้เน้นไปที่เทคโนโลยี BPMR ซึ่งถือว่าเป็นทางเลือกหนึ่งที่ช่วยเพิ่ม ความจุข้อมูลของฮาร์ดดิสก์ไดรฟ์ได้มากกว่าเทคโนโลยีการบันทึกแบบแนวตั้งที่ใช้อยู่ในปัจจุบัน 2 โดยบทนี้จะเริ่มต้นการกล่าวถึงงานวิจัยด้านต่างๆ ที่มีผลต่อการพัฒนาเทคโนโลยี BPMR จากนั้น อธิบายวิธีการสร้างผลตอบสนองสัญญาณพัลส์ของระบบ BPMR โดยอาศัยการคำนวณอินทิกรัล การตอบสนองแบบสามมิติเทียบกับศักย์ของหัวอ่านแบบ MR [108] พร้อมทังศึกษาผลกระทบ ของรูปทรงของหัวอ่านและไอแลนด์ที่มีต่อผลตอบสนองสัญญาณพัลส์แบบสองมิตินี

ในระบบการบันทึกแบบ BPMR สัญญาณอ่านกลับสร้างได้จากการซ้อนทับเชิงเส้นของ ผลตอบสนองสัญญาณพัลส์ของแทร็กหลักและแทร็กข้างเคียง จึงทำให้เกิดปัญหาเรื่องการแทรก สอดระหว่างสัญลักษณ์ (IS1) และการแทรกสอดระหว่างแทร็ก (IT1) โดยที่ ITI จะมีความรุนแรง มากขึ้น เมื่อความจุข้อมูลสูงขึ้น นอกจากนีสัญญาณรบกวนหลักที่พบมากในระบบ BPMR ก็คือ สัญญาณรบกวนสื่อบันทึกซึ่งเกิดจากความไม่สมบูรณ์ในการผลิตสื่อบันทึก (ทำให้เกิดความผันผวน ของตำแหน่ง ขนาด ความสูง และรูปร่าง ของแต่ละไอแลนด์)

ดังนั้นบทนี้จึงได้อธิบายแบบจำลองของสัญญาณอ่านกลับในระบบ BPMR ซึ่งใช้หา ค่าประมาณของสัญญาณพัลส์ BPMR (ที่มีและไม่มีสัญญาณรบกวนสื่อบันทึก) เพื่อให้สามารถ วิเคราะห์ระบบ BPMR ได้ง่ายขึ้น สุดท้ายได้กล่าวถึงปรากฏการณ์แทร็กมิสเรจิสเตรชัน (TMR) พร้อมทั้งอธิบายการสร้างสัญญาณอ่านกลับที่มีผลกระทบของแทร็กมิสเรจิสเตรชันด้วย

## 6.6 แบบฝึกหัดท้ายบท

1. จงอธิบายแนวคิดของเทคโนโลยี BPMR

2.จงอธิบายความหมายของสมการ (6.1)

3.จงอธิบายสาเหตุและผลกระทบของสัญญาณรบกวนสื่อบันทึกที่พบในระบบ BPMR

4. จงอธิบายสาเหตุและผลกระทบของแทร็กมิสเรจิสเตรชัน (TMR)

5. จงใช้โปรแกรม SCILAB วาดรูป

5.1) ผลตอบสนองสัญญาณพัลส์เกาส์เซียนแบบสองมิติตามสมการ (6.14) และ (6.15) โดย ให้กำหนดค่าพารามิเตอร์ต่างๆ ด้วยตนเอง พร้อมทั้งทดลองปรับค่าพารามิเตอร์ต่างๆ แล้ว สังเกตรูปร่างของผลตอบสนองสัญญาณพัลส์เกาส์เซียนแบบสองมิติที่ได้

5.2) สัญญาณพัลส์ตามแทร็กและสัญญาณพัลส์ข้ามแทร็กที่มีผลกระทบของ TMR ตามรูปที่ 6.30

6. กำหนดให้ t = 4 nm, W = 20 nm, g = 10 nm, a = 11 nm, δ = 10 nm, และ d = 10 nm จงใช้โปรแกรม SCILAB วาดรูป

6.1) ศักย์แม่เหล็กบนพื้นผิวของหัวอ่านแบบ MR (ตามรูปที่ 6.9)

6.2) ผลตอบสนองสัญญาณพัลส์แบบสองมิติ (ตามรูปที่ 6.13)

6.3) ผลตอบสนองสัญญาณพัลส์ตามแทร็กและโพรไฟล์ขวางแทร็ก (ตามรูปที่ 6.15)

7. ปรับค่าพารามิเตอร์ต่างๆ ที่ใช้ในแบบฝึกหัดท้ายบทที่ข้อ 6 ด้วยตนเอง แล้วทำซ้ำข้อ 6.1 – 6.3 พร้อมทั้งสังเกตและอธิบายผลลัพธ์ที่ได้ 6ล่ป น

# การออกแบบทาร์เก็ตและอีควอไลเซอร์ สำหรับระบบ BPMR

เทคนิคผลตอบสนองบางส่วนควรจะเป็นสูงสุด (PRML: partial-response maximum-likelihood) [10, 27] เป็นเทคนิคหลักที่ใช้ในการตรวจหาข้อมูลของระบบการบันทึกข้อมูลเชิงแม่เหล็กแบบ ต่างๆ เช่น แบบแนวนอน แบบแนวตั้ง และแบบ BPMR เพราะมีข้อดีคือระบบจะไม่มีปัญหาเรื่อง การขยายสัญญาณรบกวน (ทoisะ eทhaทcemeทt) และความซับซ้อนของระบบก็ไม่มากเท่าใด

วงจรตรวจหา PRML เป็นการทำงานร่วมกันระหว่างอีควอไลเซอร์และวงจรตรวจหา วีเทอร์บิ [10, 13] ซึ่งสมรรถนะของวงจรตรวจหา PRML จะขึ้นอยู่กับความเหมาะสมของทาร์เก็ต และอีควอไลเซอร์ที่ใช้ในระบบ โดยอีควอไลเซอร์จะทำหน้าที่ปรับรูปร่างของสัญญาณให้เป็นไปตาม ทาร์เก็ตที่ต้องการ จากนั้นวงจรตรวจหาวีเทอร์บิซึ่งสร้างจากทาร์เก็ตนี้จะทำหน้าที่ถอดรหัสข้อมูล นอกจากนี้การออกแบบทาร์เก็ตและอีควอไลเซอร์ที่ดีในระบบ BPMR ยังสามารถช่วยลดผลกระทบ ที่เกิดจาก ISI และ ITI ได้อีกด้วย

เนื่องจากช่องสัญญาณ BPMR มีผลกระทบจาก ITI ดังนั้นวงจรภาครับควรใช้วงจร ตรวจหาวีเทอร์บิสองมิติ (2D Viterbi detector) [108, 109] เพื่อให้ระบบมีสมรรถนะสูงสุด อย่างไร ก็ตามวงจรตรวจหาวีเทอร์บิสองมิติมีความซับซ้อนสูงมาก ดังนัน Nabav [108] และ Karakulak [109] จึงได้นำเสนอวิธีการออกแบบทาร์เก็ตและอีควอไลเซอร์แบบต่างๆ เช่น ทาร์เก็ตหนึ่งมิติกับ อีควอไลเซอร์หนึ่งมิติ, ทาร์เก็ตสองมิติกับอีควอไลเซอร์หนึ่งมิติ, และทาร์เก็ตสองมิติกับอีควอไลเซอร์ สองมิติ เป็นต้น เพื่อให้สามารถใช้วงจรตรวจหาวีเทอร์บิแบบทั่วไปได้ (นั่นคือวงจรตรวจหาวีเทอร์บิ หนึ่งมิติตามที่อธิบายในบทที่ 4 ของ [10]) เพราะฉะนั้นในบทนี้จะอธิบายวิธีการออกแบบทาร์เก็ต และอีควอไลเซอร์แบบต่างๆ สำหรับระบบ BPMR พร้อมทั้งแสดงสมรรถนะของทาร์เก็ตทั้งหมด ดังต่อไปนี้

## หมายเหตุ ในบทนี้

•ช่องสัญญาณ BPMR ที่ไม่ต่อเนื่องทางเวลาจะถูกกำหนดด้วยเมทริกซ์ H ซึ่งได้มาจากการชัก ตัวอย่าง (รampling) ผลตอบสนองสัญญาณพัลส์แบบสองมิติ ณ เวลาที่เป็นจำนวนเท่าของ คาบเวลาของบิต $( T _ { x } )$ และระยะแทร็ก (T)

สัญญาณอ่านกลับที่ได้จากหัวอ่านจะมีผลกระทบของ ISI ที่มาจากข้อมูลเพียง 3 คาบเวลาของ บิต และมีผลกระทบของ ITI ที่มาจากข้อมูลเพียง 3 แทร็กเท่านัน นั้นคือเมทริกซ์ H จะมีขนาด 3×3 (นั่นคือ 3 แถวและ 3 แนวตั้ง) ตามสมการ (6.18)

การออกแบบทาร์เก็ตและอีควอไลเซอร์ทุกแบบจะใช้วิธีการ "ข้อผิดพลาดกำลังสองเฉลี่ยน้อยสุด (MMSE: minimum mean-squared error)" [14] ซึ่งเป็นการทำให้ข้อผิดพลาดกำลังสองเฉลี่ย (MรE) ระหว่างสัญญาณที่ด้านขาออกของอีควอไลเซอร์ $\{ z _ { k } \}$ และสัญญาณด้านขาออกของ ทาร์เก็ต $\{ d _ { k } \}$ มีค่าน้อยสุด (ดูรูปที่7.1) ซึ่งวิธีการ MMSE นี้เป็นวิธีที่ง่ายและเหมาะสำหรับ การนำมาใช้งานจริงในทางปฏิบัติ
